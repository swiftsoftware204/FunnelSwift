👤 Leads

Manage your prospect leads in the Leads section. Each lead stores name, email, phone, company, source, tags, and custom fields. Leads are auto-synced to CoreSwift CRM.

Adding Leads

  1. Manual Entry

    Click "Add Lead" and fill in the form fields. Required: Name or Email. Optional: phone, company, source, tags, notes, custom fields (First Name, Last Name, Website, Address, Title, Social).

  2. Mobile App Scan

    Use the FunnelSwift Android app to scan business cards via camera. OCR parses the card and creates a lead automatically.

  3. Bulk Import

    From the mobile app, import phone contacts in bulk. Search, select, and upload multiple contacts at once.

  4. Web-to-Lead Forms

    Embed a JS snippet on your website. Any form submission auto-creates a lead in your workspace. (See Web-to-Lead section.)

  5. Kinetic Card Forms

    Add a lead capture form block to any Kinetic landing card. Visitors submit their info directly through the card.

  6. API Ingestion

    Use the REST API to create leads programmatically. Requires an API key with appropriate permissions.

Editing Leads

Click any lead to view details. In edit mode, you can change any field, add custom fields, change tags, and update notes. Stage progression: New → Contacted → Qualified → Converted → Lost.

🏷️ Tags & Tag Groups

Tags help organize leads by categories: Source, Status, Events, Services, Engagement, and Custom. Tags are grouped into tag groups for structured organization. Tags auto-sync from plan mappings and are workspace-specific.

Creating Tags

From the Tags section, you can create new tags, assign colors, and organize them into groups. Tags applied to leads in FunnelSwift are automatically pushed to CoreSwift CRM as well.

🤝 Affiliates

Manage affiliate partners. Each affiliate gets a unique auto-generated ID. Configure commission rates, track payouts, and route leads to their external systems via webhooks. Affiliate performance is visible on the dashboard.

🃏 Kinetic Cards

Kinetic Cards are server-side rendered landing pages served at /k/YOUR-SLUG. Think of them as shareable mini-sites — bio-links, digital business cards, lead capture pages, and affiliate bridge funnels — all built from editable layout blocks.

How it works: Each card is assembled from layout blocks (hero, bio, buttons, lead forms, social links, product showcase) and rendered instantly as a clean, fast HTML page. No page builder overhead. Cards are fully responsive and work great on mobile.

Creating a Kinetic Card

  1. Navigate to Kinetic Cards

    From the sidebar, click Cards under the main navigation.

  2. Choose a Template

    Click "+ New Card". Select from 4 templates: Bio-Link, Digital Business Card, Mini Funnel, or Mini Page. You can also start from scratch.

  3. Configure Content

    Fill in your content fields — name, description, image/logo URL, buttons, social links, lead form fields — depending on the template.

  4. Choose a Theme

    Select a color theme (Clean, Bold, Dark, Minimal) or use your brand colors. The card renders instantly with the chosen theme.

  5. Set Your Slug

    Give your card a unique slug (e.g. "my-link"). It will be available at https://funnelswift.net/k/my-link.

  6. Save & Share

    Save the card. Share the URL, print a QR code, or embed it anywhere. The card is live immediately.

🎯 Mini Funnel Template

The Mini Funnel template is designed specifically for affiliate marketing and product promotions. Create a focused bridge page between a traffic source (social media, ad, QR code) and your offer.

Features

  • Product Showcase — Display a product image (or video embed) with title and subtitle
  • Strong CTA — Large call-to-action button with custom text and URL
  • 4 Themes — Clean (light, professional), Bold (vibrant, high-contrast), Dark (sleek), Minimal (ultra-clean)
  • Mobile-First — Optimized for mobile traffic from social media and ads
  • Lead Capture — Optional embedded form before the CTA
Use case: Mini Funnels are ideal for Facebook/Instagram ads, influencer shoutouts, and QR code campaigns. Instead of sending traffic to a generic landing page, create a branded bridge page that leads to your offer. The 4 themes help match your ad creative.

Setting Up a Mini Funnel

  1. Create Card

    Click "+ New Card" and select the "Mini Funnel" template.

  2. Product Image or Video

    Upload a product image URL or paste a video embed URL (YouTube/Vimeo). One or the other — pick what converts best.

  3. Write Copy

    Add a compelling title and subtitle that drives the visitor to click the CTA.

  4. Set CTA

    Enter the button text (e.g. "Get My Free Trial", "Claim Discount") and the destination URL.

  5. Pick Theme

    Choose from Clean, Bold, Dark, or Minimal — whichever matches your brand or ad creative.

  6. Enable Lead Form

    Optionally add a name + email form before the CTA to capture leads before the click.

  7. Publish

    Save and share. Your Mini Funnel is live at /k/YOUR-SLUG.

Pro tip: Use different slugs for different traffic sources (e.g. /k/insta-offer, /k/email-blast, /k/qr-poster) so you can track which source converts best via Insights.

📈 Insights

The Insights section gives you analytics on your Kinetic cards. Track views, lead captures, click-through rates, and conversion performance over time.

Available Metrics

  • Card Views — Total visits to each card, unique vs returning
  • Lead Conversions — How many visitors submitted the lead form
  • CTA Clicks — Button click tracking for each CTA on the card
  • Top Cards — Your best-performing cards by views and conversions
  • Traffic Sources — Where visitors are coming from (direct, social, QR, referral)

Viewing Insights

From the sidebar, click Insights. Select a card from the dropdown or view all cards aggregated. Data is updated in near real-time and can be filtered by date range.

📲 Sharing the App

Share FunnelSwift with someone without asking for their phone number:

  • Install as App (PWA) — Visit funnelswift.net on their phone. An Install bar appears at the bottom — tap it to add FunnelSwift to the home screen. No app store needed.
  • Short link — kntcrd.com/app redirects straight to the download page. Easy to say out loud.
  • AirDrop / Nearby Share — Share the URL directly with someone nearby.
  • NFC stickers — Buy blank programmable NFC tags, write funnelswift.net to them, and tap the tag to their phone.

📱 QR Codes

Each Kinetic card can be paired with a QR code for print and physical-world use. QR codes are generated server-side as SVG (for print) or PNG (for digital).

Generating QR Codes

  1. Open a Card

    From the Cards list, click the QR icon on any card.

  2. Configure

    Choose size (small/medium/large) and format (SVG for print, PNG for digital).

  3. Download

    Download the QR code image. It links to your card's URL by default.

  4. Print or Share

    Use on flyers, posters, business cards, packaging, or in-store displays.

Plan limits: QR code generation limits depend on your plan. Free plans have a monthly limit. Enterprise plans include unlimited QR code generation.

QR Code API

For programmatic generation, use the QR code API endpoint:

GET /api/v1/kinetic/qr/{slug}?format=svg&size=300
GET /api/v1/kinetic/qr/{slug}?format=png&size=500

🌐 Web-to-Lead

Web-to-Lead lets you capture leads from any external website form. Embed a small JavaScript snippet on your site, and FunnelSwift will automatically ingest form submissions as leads.

How it works: Your user copies a snippet, pastes it on their website, and every form submission is captured and routed to their FunnelSwift workspace — tagged, deduplicated, and ready to use.

Setting Up a Web-to-Lead Config

  1. Navigate to Web-to-Lead

    From the sidebar, click Web-to-Lead.

  2. Create a Config

    Click "+ New Config". Give it a name (e.g. "Website Contact Form"), set a default source label, optionally assign one or more tags for auto-labeling, and set a rate limit (default 100/hr).

  3. Get the Embed Code

    Click the link icon (🔗) on your config. A modal shows the embed code and instructions.

  4. Create an API Key

    Go to API Keys and create a new key. Copy the full key.

  5. Paste on Your Website

    Copy the embed code, replace YOUR_FULL_API_KEY_HERE with your actual key, and paste it just before </body> on your website.

The Embed Snippet

<!-- FunnelSwift Web-to-Lead -->
<script>
window.FunnelSwiftConfig = {
  apiKey: "fs_live_xxxxxxxxxxxxxxxx",
  configId: "00000000-0000-0000-0000-000000000000"
};
</script>
<script src="https://funnelswift.net/funnelswift-capture.js" async></script>

Smart Field Mapping

The JavaScript snippet automatically detects form fields by name and label. It maps common patterns like email, name, phone, company, message, and more. Unrecognized fields pass through as custom data.

Duplicate Detection

If a submission has an email that already exists, the system returns success but does not create a duplicate lead. Duplicates are logged for your reference.

Rate Limiting

Default 100 submissions per hour per workspace. Configure per config under Web-to-Lead settings.

Direct API Endpoint

POST https://funnelswift.net/api/v1/web-to-lead
Content-Type: application/json

{
  "api_key": "fs_live_xxxxxxxxxxxxxxxx",
  "config_id": "00000000-0000-0000-0000-000000000000",
  "name": "John Smith",
  "email": "john@example.com",
  "company": "Acme Inc",
  "phone": "+1-555-0123"
}

🔑 API Keys

Generate API keys for programmatic access to FunnelSwift. Keys are hashed with Argon2 for security. Permissions control what each key can do, including web-to-lead access.

Creating an API Key

  1. Go to API Keys from the sidebar.
  2. Click "+ New Key" and give it a descriptive name.
  3. Copy the key immediately — it's shown once.
  4. Use with Web-to-Lead by pasting it into your embed snippet's apiKey field.
